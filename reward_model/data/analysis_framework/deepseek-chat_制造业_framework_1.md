# 制造业企业业务驱动财务分析框架

## 一、业务表现全景扫描

**制造效能与市场响应度分析**
- 产能利用率 vs 订单交付率
- 产品良率与返修成本率
- 客户订单准时交付率
- 新产品收入贡献度
- 核心客户留存率与价值增长

### （2）价值链效率追踪
**从原材料到现金回收的全周期分析**
- 原材料库存周转 vs 产成品库存周转
- 供应商账期 vs 客户账期匹配度
- 制造周期时间优化趋势
- 单位制造费用变动分析
