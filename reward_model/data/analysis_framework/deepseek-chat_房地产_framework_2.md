# 房地产企业财务风险分析框架

## 一、风险态势总览

### （1）风险等级评估
- 综合风险评分及等级划分（高风险/中风险/低风险）
- 关键风险指标雷达图展示
- 风险变化趋势（月度环比、年度同比）

### （2）风险预警信号
- 红色预警指标（需立即关注）
- 黄色预警指标（需持续监控）
- 风险传导路径分析

## 二、现金流断裂风险分析

### （1）现金安全边际
- 现金覆盖倍数（可维持运营月数）
- 经营性现金流与到期债务匹配度
- 预售资金监管账户余额分析

### （2）融资渠道稳定性
- 银行授信使用率及续期情况
- 债券到期分布及再融资能力
- 非标融资占比及成本变化

### （3）项目现金流平衡
- 单个项目现金流回正周期
- 项目间资金调配能力
- 土地款支付压力测试

## 三、债务风险穿透分析

### （1）债务结构风险
- 短长债比例及期限错配分析
- 表外负债风险敞口评估
- 明股实债规模及影响

### （2）偿债能力压力测试
- 不同销售去化率下的偿债能力
- 房价波动对抵押物价值的影响
- 再融资环境变化的敏感性分析

### （3）融资成本管控
- 综合融资成本率变化
- 利率风险敞口管理
- 汇兑风险（如有外币负债）

## 四、经营风险敞口评估

### （1）去化风险分析
- 存货去化周期（按业态、区域细分）
- 预售许可证获取进度风险
- 客户退房率及原因分析

### （2）成本超支风险
- 建安成本变动对毛利率的影响
- 土地增值税清算风险
- 开发间接费用控制有效性

### （3）合作开发风险
- 合作方信用风险评估
- 并表/非并表项目风险隔离
- 合作纠纷潜在影响

## 五、资产质量与流动性分析

### （1）资产变现能力
- 存货分类质量评估（优质/一般/问题资产）
- 投资性房地产公允价值变动
- 持有型物业租金收益率

### （2）资产周转效率
- 存货周转天数（按开发阶段细分）
- 应收账款回收风险
- 受限资产比例及影响

### （3）土地储备质量
- 土地成本与市场价值比较
- 土储去化年限合理性
- 重点城市土地储备集中度

## 六、盈利能力韧性测试

### （1）毛利率抗风险能力
- 各业态毛利率波动分析
- 价格调整对盈利的敏感性
- 成本管控的边际效益

### （2）费用刚性分析
- 固定费用占比及压缩空间
- 营销费用投入产出效率
- 管理费用的人均效能

### （3）利润质量评估
- 主营业务利润占比
- 资产处置等非经常性损益影响
- 合作项目收益确认风险

## 七、行业对标与压力测试

### （1）关键指标行业分位值
- 资产负债率、净负债率行业对比
- 现金短债比、扣预资产负债率对标
- 毛利率、净利率行业位置

### （2）极端情景压力测试
- 销售下滑30%情景下的现金流
- 融资成本上升200BP的影响
- 土地价格下跌20%的资产减值

### （3）监管政策敏感性
- 预售资金监管政策变化影响
- 贷款集中度管理要求的影响
- 土地出让政策调整的适应性

## 八、风险应对与改进建议

### （1）短期风险处置方案
- 现金流缺口弥补的具体措施
- 高成本债务置换计划
- 资产处置回款时间表

### （2）中期风险防控策略
- 债务结构优化目标及路径
- 存货结构改善的具体方案
- 融资渠道多元化建设

### （3）长期韧性提升规划
- 资产质量改善的量化目标
- 盈利能力稳定性的提升措施
- 风险预警体系的完善方案

### （4）情景模拟与效果预估
- 不同改进方案的风险降低效果
- 投入产出比的量化分析
- 实施时间表及里程碑设置
