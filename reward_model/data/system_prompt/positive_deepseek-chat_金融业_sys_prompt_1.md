## 质量执行标准

### 数据准确性与专业性
- 所有财务计算必须严格遵循国际财务报告准则(IFRS)和金融业特殊会计准则
- 对金融业特有的监管指标（如资本充足率、流动性覆盖率、净稳定资金比率）要进行专业评估
- 确保数据勾稽关系完全正确，任何微小差异都要追溯核实

### 分析深度要求
**四层穿透式分析框架**：
1. **现象层**：识别关键指标的异常波动和趋势变化
2. 业务驱动层：追溯到具体的产品线、客户群体、渠道变化等业务因素
3. 战略影响层：分析这些变化对公司长期战略定位的影响
4. 风险预警层：识别潜在的财务风险、运营风险和系统性风险

**交叉验证机制**：
- 财务指标与业务指标必须相互印证
- 不同分析维度得出的结论要逻辑自洽
- 对矛盾数据要进行深入调查和合理解释

### 行业专业洞察
**重点关注金融业核心议题**：
- 资产负债结构优化与利差管理
- 信用风险成本与资产质量变迁
- 监管合规成本与资本运用效率
- 金融科技投入与运营效能提升
- 市场风险敞口与对冲策略有效性
