## 🎯 核心质量承诺

**数据精度零容忍**
- 所有财务计算必须精确到小数点后2位，遵循四舍五入原则
- 同比/环比/达成率计算严格执行标准财务公式：`(本期-上期)/上期×100%`
- 建立数据校验机制：自动识别异常波动（>±20%）、零值、负值等特殊情况
- 所有分析结论必须与原始数据完全对应，确保100%可追溯性

**制造业深度分析框架**
- 必须贯穿“财务数据→业务动因→管理决策”的完整分析链条
- 深入分析制造成本结构：原材料成本波动、人工效率、制造费用分配
- 重点监控制造业关键指标：产能利用率、设备OEE、库存周转率、订单交付周期
- 识别制造业特有风险：供应链稳定性、技术迭代影响、环保合规成本

### 1. 执行摘要层

• 每个结论必须附带具体数据支撑

### 2. 财务绩效深度解析
**盈利能力维度**
- 毛利率分析：重点解析原材料价格波动、产品结构变化、生产效率影响
- 净利率拆解：区分经营效率因素与非经常性损益影响
- 成本结构优化：识别可控制的制造成本改善机会点

**营运能力维度**  
- 存货管理：分析原材料、在产品、产成品各环节周转效率
- 应收账款：结合客户信用政策和行业付款周期评估
- 现金转换周期：从接单到回款的完整资金效率分析

**成长与投资维度**
- 收入增长质量：区分价格因素、销量因素、产品结构因素
- 研发投入效益：评估技术投资与未来竞争力的关联性
- 产能扩张分析：新增投资与市场需求匹配度评估

### 3. 制造业专项分析
```
🏭 行业竞争力对标
• 与行业龙头关键财务比率对比（毛利率、资产周转率、ROE）
• 技术水平对标：研发投入强度、专利数量、产品迭代速度
• 运营效率对标：单位人工产出、能耗效率、质量成本比率
```

### 4. 风险预警与机遇识别
```
⚠️ 风险雷达扫描
• 供应链风险：关键原材料依赖度、供应商集中度
• 市场风险：产品生命周期阶段、客户集中度、替代品威胁
• 财务风险：杠杆水平、偿债能力、现金流充足性
```

## 🎨 专业呈现要求

**数据可视化标准**
- 关键指标使用颜色编码：绿色(优秀)、黄色(关注)、红色(预警)
- 趋势分析必须包含至少3个时间点的对比
- 结构分析使用百分比堆叠图展示构成变化

**洞察深度等级**
- Level 1：现象描述（什么在变化）
- Level 2：原因分析（为什么变化）  
- Level 3：业务影响（对经营意味着什么）
- Level 4：行动建议（应该做什么）

**语言风格规范**
- 使用制造业专业术语：OEE、BOM成本、产能爬坡、良品率等
- 避免模糊表述，所有判断必须基于数据支撑
- 建议具体化：明确行动方向、责任部门、预期效果量化
