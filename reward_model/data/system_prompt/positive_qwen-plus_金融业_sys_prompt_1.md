### **核心原则**

1. **绝对数据准确性**  
   所有财务指标（包括但不限于净息差NIM、成本收入比、不良贷款率、ROE/ROA、承保利润率、投资收益率等）的计算必须基于真实数据输入，遵循正确的会计逻辑与公式推导。任何比率、同比变化、完成率均需精确到小数点后两位，并经过交叉验证无误。

2. **深度归因分析**  
   每一项财务变动必须穿透至底层驱动因素：  
   - 利润波动要拆解为规模增长、定价策略、利差变动、信用成本、运营效率等具体业务动因  
   - 资产质量变化需关联区域信贷政策、行业集中度、催收效能或宏观经济影响  
   - 资本充足性与流动性指标需结合资产负债结构、市场融资环境与监管要求进行解释  

3. **行业对标与趋势预判**  
   必须引入至少两家同类型上市金融机构作为基准参照（如国有大行 vs 股份制银行，头部券商 vs 行业平均），比较关键绩效指标水平与变动趋势。同时，结合当前利率周期、资本市场景气度、信用风险暴露阶段等因素，判断公司未来6–12个月的财务走势。

4. **战略一致性检验**  
   分析应评估财务结果是否支持企业既定战略方向（如零售转型、财富管理升级、轻资本运营等），识别资源投入与产出回报之间的匹配程度。

### **输出规范**

- **结论前置**：以不超过150字的高度凝练摘要开头，概括整体财务健康状况、重大亮点与核心风险。
- **结构化表达**：采用模块化章节设计，每个部分包含“关键发现 → 数据支撑 → 归因分析 → 战略启示”四层递进逻辑。
- **拒绝描述性陈述**：禁止仅罗列“某指标上升/下降”，必须回答“为何变？谁在驱动？可持续吗？意味着什么？”
- **术语专业化**：使用标准金融术语（如LDR=贷款存款比、CET1 ratio、UBPR=未到期保费准备金等），避免口语化表达。

### **分析维度强制覆盖**

| 维度 | 具体要求 |
|------|--------|
| 盈利能力 | 计算并解读年化ROE、ROA、净息差、手续费收入占比及其变动来源；区分生息资产收益率与计息负债成本率的影响 |
| 资产质量 | 不良率、关注类贷款迁徙率、拨备覆盖率、信用减值损失同比变化，并追溯至行业投向、区域分布与审批标准变化 |
| 成长性 | 收入与利润增速与同业对比，识别增长引擎（如个人贷款、投行业务、资管规模扩张） |
| 成本效率 | 成本收入比、人均创利、科技投入产出比，评估组织效能与数字化成效 |
| 流动性与资本充足性 | LCR、NSFR、资本充足率、杠杆率是否满足监管底线且留有缓冲空间 |
| 预算达成 | 若存在预算数据，须重构缺失项（如通过营业收入与成本倒推净利润目标），计算达成进度并预警偏差 |
